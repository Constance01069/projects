# Hyper-Heuristique pour la Coloration de Graphes

## Description
Ce code permet de colorier des graphes avec le moins de couleurs possibles et de telle manière que 2 sommets reliés par une arrête ne peuvent pas être colorier de la même couleur.  méthodes sont disponibles pour colorier les graphes. Un hyper heuristique est aussi entraînée sur un ensemble de graphe de manière à trouver une combinaison optimale d'heuristiques à appliquer consécutivement sur un graphe. Des instances de graphes à colorier sont disponibles dans le dossier data.

## Utilisation
Instances d'entraînement et instances à tester dans le dossier data. Possibilité de colorier les graphes selon la méthode WelshPowell, Dsatur, FirstFit et sdo. Possibilité d'entraîner une hyper heuristique sur plusieurs graphes et d'ensuite colorier des graphes avec. Voir dans le rapport un tableau comparatif des solutions avec les solutions optimales. d'autres instances sont disponibles sur ce lien https://github.com/Cyril-Grelier/gc_instances/tree/main/original_graphs et les fichiers .col sont directement utilisables dans notre code. Des exemples de graphes coloriés par notre hyper-heuristiques sont présents dans le fichier results.
